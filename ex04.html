if-sats

Ibland måste program kunna ta beslut utifrån information.

ex a:

// kör följande kodblock om uttrycket i parantes evalueras som sant:
if (true) {
  Console.WriteLine("Tjena!");
}
// ignorera följande kodblock om uttrycket i parantes evalueras som falskt:
if (false) {
  Console.WriteLine("Tjena!");
}

ex b:

If-satsen kan utökas med else if: 
if (false) {  // detta kodblock ignoreras
  Console.WriteLine("A");
} else if (true) {  // detta kodblock körs
  Console.WriteLine("B");
}
